### 아두이노 기본 부품

- 저항 : 직렬연결을 하여 전압 분배를 통해 소자들이 갖는 전압 값을 안정화
    - 첫 번째 빨간색 띠는 숫자 2를 의미
    - 두 번째 빨간색 띠도 숫자 2를 의미
    - 세 번째 띠는 검은색으로 숫자 0을 의미
    - 네 번째 띠는 배수를 의미하며, 검은색이므로 * 1 배수
    - 마지막으로 제일 오른쪽 띠는 +-1%(저항의 오차)를 의미

- 브레드 보드 : 전자 회로 실험에서 기판에 납땜을 하지 않고도 소자들을 연결할 수 있도록 사용하는 회로 구성 도구

- VCC : +
- GND : -

- 시리얼 통신 : 아두이노의 통신 방식. 숫자를 하나씩 전달하지만 더 멀리 보낼 수 있음(직렬 통신)

- 아두이노 간단 문법
    - HIGH = 점등, LOW = 소등
    - int - 정수, float - 실수, char - 문자(”큰따옴표로 표현”)

```jsx
a = 7;
b = -5;

Serial.println("ab");     // ab
Serial.println(ab);       // 2
Serial.println(a-b);      // 12
Serial.println(a*b);      // -35
Serial.println(a/b);      // -1
```
