# 7장


### 디지털 데이터 입출력

- pinMode, digitalRead, digitalWrite, analogRead, analogWrite

|  | 출력 | 입력 |
| --- | --- | --- |
| 디지털 | pinMode(13, OUTPUT); digitalWrite(13, HIGH) | pinMode(13, INPUT)
boolean state = digitalRead(13) |
| 아날로그  | analogWrite(3, 128); | int v = analogRead(A0); |

- 아두이노 우노 기준 20개 데이터 핀으로디지털 데이터 입출력은 모두 가능
- 아날로그 데이터 입출력은 일부 핀을 통해서만 가능

- 풀업 저항 : 스위치가 개방된 상태에서 핀에 가해지는 값이 VCC가 되도록 끌어 올리는 역할을 하는 저항
    - 풀업 저항 상태에서 버튼을 누르면 논리 0(GND) 누르지 않았을 때 논리 1(VCC) 리턴

- 풀다운 저항 : 스위치가 개방된 상태에서 핀에 가해지는 값이 GND가 되도록 끌어 내리는 역할을 하는 저항
    - 풀다운 저항 상태에서 버튼을 누르면 논리 1(VCC), 누르지 않았을 때 논리 0(GND) 리턴
    
    ![Untitled](https://prod-files-secure.s3.us-west-2.amazonaws.com/2047c6cb-f04d-4bf3-9128-dee1685eda2d/d39b6340-5de0-4f17-a227-d92f71f07cad/Untitled.png)
    
- 버튼을 누른 횟수 세기
    - 채터링 현상 OR 바운스 현상 : 버튼을 한번 눌렀을 때 짧은 시간 동안 논리 1과 논리 0 사이를 진동하고 두 번 이상 눌린 것으로 인식되는 현상
    - 디바운스 : 채터링 현상을 없애는 방안
        - 소프트웨어적 디바운스 : 짧은 시간 내에 두 번 검사하여 버튼이 완전히 눌린 것을 확인
        - 하드웨어적 디바운스 : 커패시터 사용
    

아날로그 데이터 입력

- 아두이노 우노 ATmega328에는 6채널 ADC(AnalogDigitalConverter)가 포함됨
- A0에서 A5까지의 상수 변수가 정의되어 있음
- 디지털 핀 번호 ‘14’나 상수 변수 ‘A0’ 어느 것을 사용해도 무방
- ATmega328의 ADC는 10비트의 해상도를 가짐
- 6개 채널은 하나의 ADC에 멀티플렉서로 연결되어 있어 한번에 하나의 아날로그 입력만을 디지털로 변환할 수 있음

아날로그 데이터 출력

- 디지털 데이터를 아날로그 데이터로 변환하기 위해 디지털 - 아날로그 변환기가 필요함. 하지만, 아두이노 우노는 DAC가 포함하지 않음
- 대신 아날로그 신호와 유사한 효과를 내는 펄스 폭 변조(PWM, Pulse Width Modulation) 신호를 사용 가능
- PWM 신호 출력이 가능한 핀은 디지털 핀 중에서 물결무늬가 표시된 핀

- 펄스 진폭 변조(PAM, Pulse Amplitude Modulatior)
    - 샘플링 : 일정한 시간 간격으로 아날로그 신호의 값을 취하는 과정
    - 양자화 : 샘플링된 신호는 주어진 해상도에 따라 가장 가까운 디지털 값으로 바꾸는 과정
    - 양자화된 신호를 0과 1의 비트열로 나타내어 출력

- 펄스 폭 변조(PWM, Pulse Width Modulation)
    - 양자화된 값을 0과 1의 비트열로 나타내는 PAM방식과 달리 펄스의 폭으로 나타내는 차이가 있음
    - 듀티 사이클 : 신호가 한 주기 내에서 갖는 HIGH 값의 비율
    - PWM 장점
        - PAM신호에 비해 간단하게 아날로그 신호로 변환
        - 그 자체로도 아날로그 신호의 효과
    - PWM 단점
        - PAM 방식에서 하나의 샘플을 표현하기 위해서는 8비트면 충분, 하지만 PWM 방식에서는 255비트가 필요
        - PWM 방식으로 디지털 데이터를 저장하기 위해서는 많은 저장 공간이 필요하며, 동작 주파수 역시 PAM과 비교했을 때 훨씬 높음
