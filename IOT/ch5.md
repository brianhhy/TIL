### UART 시리얼 통신

- UART 시리얼 통신 : 데이터를 보내거나 받는 하드웨어를 가르키는 말로, 전용 하드웨어를 사용하여 이루어지는 시리얼 통신을 가리키는 말
    - RXD와 TXD, 전원을 위한 VCC와 GND 최소 4개가 필요함
    - 아두이노 우노는 하나의 하드웨어 시리얼 포트만 제공 : 0번(RX)과 1번(TX)

보율 : 변조 속도를 나타내는 단위로, UART 통신에서 보내는 쪽과 받는 쪽에서 데이터의 송수신 속도를 나타냄

UART 통신에서는 ‘0’의 시작 비트와 ‘1’의 정지 비트를 사용하고 바이트 단위 통신을 주로 사용하여, 총 10비트 단위로 데이터를 전송함

Serial은 HardwareSerial클래스의 유일한 객체로 미리 선언되어 있으므로, 별도로 객체를 생성할 필요 없이 바로 사용 가능

복잡한 이유 : UART 통신이 비동기식 통신

Begin(Unsigned long baud)
Print(value, format) : 문자열 기반 데이터
Write(unit8_t, ch) : 이진 데이터 기반 출력
BIN : 2진수
OCT : 8진수
DEC : 10진수(생략 가능)
HEX : 16진수
