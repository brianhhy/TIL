# 3장


## 아두이노 우노 보드

---

### 전원 공급

- USB 케이블로 컴퓨터와 연결(가장 간단한 방법)
- 배럴 잭을 통해 직류 전원에 연결
- Vin 핀 헤더를 통해 연결
- VCC 핀 헤더를 통해 연결

### 핀 헤더

- 아두이노 우노는 32개의 피메일 핀 헤더를 제공
- 14핀(0부터 13까지)은 데이터 입출력, 6핀(A0부터 A5)은 아날로그 데이터 입력, 6핀은 전원 관련 핀, 나머지 6핀은 RESEST, IOREF, AREF, SCL, SDA, nc등이 있다.
- 마이크로 컨트롤러에서 모든 데이터 핀은 디지털 데이터 입출력에 사용 가능. A0 ~ A5대신 14 ~ 19번 사용가능
- 데이터 핀 하나에 공급할 수 있는 최대 전류는 40mA
- 20개 데이터  전체를 통해 공급할 수 있는 전류 합은 200mA를 넘을 수 없음
- 핀의 추가 기능은 시리얼 통신과 외부 인터럽트 기능

### 마이크로 컨트롤러

- ATmega 328 : 아두이노 우노의 핵심 마이크로 컨트롤러
- ATmega16u2 : 스케치 업로드 및 컴퓨터와의 UART 시리얼 통신을 위해 사용되는 USB - UART 변환 전용 마이크로 컨트롤러. USB 연결을 가능하게 함.

### LED

- TX LED : ATmega328에서 시리얼 통신을 통해 데이터를 보내는 경우 확인
- RX LED : 시리얼 통신을 통해 ATmega328이 데이터를 받는 경우 확인
- 13번 핀의 LED : ‘Blink’ 예제의 테스트용 LED, 마이크로 컨트롤러의 부팅 과정과 프로그램 업로드 시작 시 확인

### 아두이노 개발 설정

- setup과 loop라는 2개의 기본함수를 가지고 있어야 함
